CSS border-radius - Specify Each Corner
Four Values: first value applies to top-left corner, second value applies to top-right corner, third value applies to bottom-right corner, and fourth value applies to bottom-left corner.
Three Values:first value applies to top-left corner, second value applies to top-right and bottom-left corners, and third value applies to bottom-right corner.
Two Values:first value applies to top-left and bottom-right corners, and the second value applies to top-right and bottom-left corners.
One Value:the value applies to all four corners, which are rounded equally.
